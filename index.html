About

Bit Crate Token (BCT) powers the Bit Crate ecosystem on Polygon for membership utilities, airdrops, quests, and seasonal events. The token is fixed supply (100M), non-upgradeable, and has no transfer tax. Ownership is held in a Safe multisig for security and transparency.
